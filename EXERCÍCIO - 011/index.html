Para usar a tag pictures, "'picture e depois 'media'", aumentar e diminuir tamanho

Explicação

Tem que colocar a tag picture, e dentro dela vai os três tamanhos de imagens, a imagem principal é posta usando a tag img, e as outras duas são postas usando a tag source-media-type. O media colocamos max-width: 1050px, e o type é imagem/png

Colocando áudios

Usar a tag audio, depois controls e depois autoplay
